# nmearouter [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

## Install

```sh
$ npm install --save nmearouter
```


## Usage

```js

nmearouter('Rainbow');
```
